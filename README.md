## 🕵️ Prop Hunt — Multiplayer Browser Game

A real-time 2D top-down multiplayer Prop Hunt game built with:
- **Next.js 14** (web frontend)
- **Phaser 3** (game engine)
- **Colyseus** (authoritative multiplayer server)
- **TypeScript** everywhere
- **Tailwind CSS** (UI)

## Architecture
```
prop-hunt/
├── apps/web        → Next.js + Phaser 3 client
├── apps/server     → Colyseus authoritative server
└── packages/shared → Shared types, constants, map data
```

## Prerequisites

- Node.js 18+
- npm 9+

## Quick Start
```bash
# 1. Install dependencies
npm install

# 2. Copy environment config
cp .env.example apps/web/.env.local

# 3. Start everything
npm run dev
```

Then open two browser tabs to `http://localhost:3000`:
1. Tab 1: Create a room → copy the room code
2. Tab 2: Join with the room code
3. Both players ready up → game starts!

## How to Play

### Props 🎭
- Move with **WASD** or **Arrow Keys**
- Press **F** near an object to disguise as it
- Press **F** again to revert
- Survive until the timer runs out!

### Hunters 🏹
- Move with **WASD** or **Arrow Keys**
- Press **E** to attack/check nearby props
- Attack has a 1.5s cooldown
- Find and eliminate all props before time runs out!

## Match Flow

1. **Lobby** — Players join, see the list, ready up
2. **Countdown** — 3s before hiding phase
3. **Hiding Phase** — 30s for props to find hiding spots
4. **Hunting Phase** — 3 minutes to find all props
5. **Round End** — Scores shown, rematch option

## Development
```bash
# Server only
cd apps/server && npm run dev

# Web only  
cd apps/web && npm run dev

# Both (from root)
npm run dev
```

## Deployment

### Server (Railway / Render / Fly.io)
```bash
cd apps/server
npm run build
npm start
```
Set `PORT` environment variable.

### Web (Vercel)
```bash
cd apps/web
npm run build
```
Set `NEXT_PUBLIC_COLYSEUS_URL=wss://your-server-domain.com`

## Room Codes

Room IDs are generated by Colyseus. Share the URL or the room code displayed in the lobby.
